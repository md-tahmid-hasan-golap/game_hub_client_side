# Gamehub - A Game Library

## Description

**Gamehub** is an engaging online library for discovering and supporting game developers. Users can browse indie games, see detailed information, and install games they like. The platform offers authentication, dynamic game details, responsive design, animations, and a newsletter subscription to enhance user engagement.

## Key Features

- Fully responsive design for mobile, tablet, and desktop.
- Authentication using Email/Password and Google login (Firebase).
- Protected routes for game details and user profile.
- Update user information (Name & Profile Picture).
- Dynamic tab titles for each page.
- Popular Games section with ratings sorting.
- Banner slider with multiple slides.
- Newsletter subscription section.
- Forgot password functionality.
- 404 Not Found page.
- Extra route based on website theme.
- Smooth animations using **Motion**.
- Sweet alerts for user interactions.
